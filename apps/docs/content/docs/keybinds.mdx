Press `?` at any time to see keybinds in the built-in help overlay.

## Global

| Key | Action |
|-----|--------|
| `0` | Focus preview pane |
| `1` | Focus session list |
| `o` | Switch to tmux pane |
| `O` | Attach session in popup |
| `v` | Enter copy mode |
| `?` | Toggle help overlay |
| `q` | Quit |
| `Ctrl+C` | Quit |

## Preview Pane

| Key | Action |
|-----|--------|
| `/` | Search forward |
| `?` | Search backward |
| `y` | Copy all content to clipboard |
